## html теги 
> `<br>` - с новой строки 
> `<ul>` устанавливает маркированный список. Каждый элемент списка должен начинаться с тега `<li>`. Если к тегу `<ul>` применяется таблица стилей, то элементы `<li>` наследуют эти свойства   

## События
> onclick - нажать
> onmouseover - при навидении
